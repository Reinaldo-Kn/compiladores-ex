## Compilação

Para compilar qualquer um dos códigos, utilize o comando `g++`:

```bash
g++ codigo.cpp -o codigo
```

### Exercício 1
```bash	
./lexico codigo.c
```

### Exercício 2
```bash
./parser "expressao"
``` 
exemplo:
```bash
./parser "i=(io(ieni))"
```

### Exercício 3
```bash
./parser entrada
```
exemplo:
```bash
./parser nn+
```

## Ambiente Windonws
Para usuários Windows, é recomendado o uso do Windows Subsystem for Linux (WSL) para compilar e executar esses códigos.
